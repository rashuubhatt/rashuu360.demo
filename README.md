# Hotel-Food-Ordering_System
A python based Hotel management project where customers can view the avilable items ,& add items to their order ,see the total price and confirm their order .
<br>
This project focuses on basic logic building ,user intereactions and real world flow of simple hotel ordering system.

<br>

## Features:
- Display avilable food menu
- Add items to the Order
- calculate total bill Automatically
- confirm costumer order 

<br>

## Technologies Used:
* python
* Basic programming concepts
* conditional statement
* loops
* functions
